#Console Games Store

This is my first web project uploaded on GitHub   
It is a **simple console games store** (PlayStation, Xbox) where users can browse games and view basic details.

## Project Goals
- Practice creating a web project and using GitHub.
- Design a simple store layout for showcasing games.
- Learn the basics of HTML, CSS, and JavaScript.

## How to Run
1. Download the project (ZIP) or clone the repository.
2. Open the `index.html` file in your browser.
3. Explore the store sections and check out the listed games.

##Notes
- This project is for **practice only** (no real shopping system included).
- Future improvements may include features like search or a shopping cart.
